How to Play

Use your mouse to move the basket at the bottom of the screen. Catch the falling balls that match the current target color.

Game Rules

  • Each correct catch: +10 points
  • Wrong color catch: -5 points
  • Game stops automatically if score drops below -20
  • Target color changes randomly during gameplay

Controls

Start: Begin the game

Try Again: Appears after game over to restart
